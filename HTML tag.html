1. html tag: The root element of an HTML document. It represents the entire HTML content
  2. head tag: Contains information about the HTML document, such as title, metadata, and CSS stylesheets
  3. title tag: Specifies the title of the HTML document that appears in the browser's title bar or tab
  4. body tag: Encloses the main content of the HTML document, including text, images, links, and other elements
  5. header tag: Represents the header section of a webpage, typically containing a site logo, site navigation, or introductory content
  6. nav tag: Defines a set of navigation links for a webpage. Usually used for menus or navigation bars
  7. section tag: Represents a standalone section of content within a webpage, often with its own heading or topic.
  8. article tag: Defines an independent, self-contained content piece within a webpage like a blog post, news article, or product description
  9. aside tag: Contains tangentially related content, separate from the main content, such as sidebars or callouts
  10. footer tag: Represents the footer section of a webpage, usually containing copyright information, contact details, or related links.
