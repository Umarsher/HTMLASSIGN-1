HTML
HTML (Hypertext Markup Language) is the standard markup language for creating the structure and content of web pages.
CSS
CSS (Cascading Style Sheets) is used for describing the look and formatting of a document written in HTML.
JavaScript
JavaScript is a high-level, interpreted programming language that is widely used for adding interactivity and dynamic behavior to web pages.
Node.js
Node.js is a runtime environment that allows executing server-side JavaScript code. It enables building scalable network applications.
Express
Express is a fast and minimalist web application framework for Node.js. It simplifies building web applications and APIs.
React
React is a JavaScript library used for building user interfaces. It allows for creating interactive components and efficient UI updates.
Angular
Angular is a TypeScript-based framework for developing dynamic web applications. It provides a comprehensive set of tools for building complex solutions.
Vue.js
Vue.js is a progressive JavaScript framework for building user interfaces. It emphasizes simplicity and is often referred to as the "reactive framework".
MongoDB
MongoDB is a NoSQL database used for storing and retrieving large amounts of data. It provides high scalability and flexibility.
MySQL
MySQL is a widely used open-source relational database management system. It is known for its reliability, performance, and ease of use.
PostgreSQL
PostgreSQL is an advanced open-source object-relational database system. It offers extensibility, SQL compliance, and strong data integrity.
Git
Git is a distributed version control system used for tracking changes in source code. It enables coordination and collaboration among developers.
